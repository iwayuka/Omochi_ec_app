# OMOCHIYA

## サイト概要
お餅屋さんのECサイト。

### サイトテーマ
お餅のネット販売と製造に必要な管理を行うサイト。

### テーマを選んだ理由
お餅屋さんのECサイトを制作したいと思いました。
ネット販売で売上を伸ばす手伝いや、
発注から必要な材料などを計算でいると便利なのではないかと思い、このテーマにしました。


### ターゲットユーザ
管理者：小さなお餅加工場。
　　　　小さい文字は読みづらい。パソコンを普段使わない。
　　　　シンプルなデザインで、見たい対象をすぐ見れるように注意する。
会　員；お餅が大好きな方にネットでお買い物ができるように。
　　　　文字は大きく、シンプルなデザインを心がける。

### 主な利用シーン
管理者：生産管理（お餅の販売/受注管理）
会　員：お餅の購入
